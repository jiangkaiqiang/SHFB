创意集装箱活动房屋之游牧博物馆


这间PROYECTO Container工作室位于巴西的圣保罗，由H²O Arquitetura建筑设计事务所设计。

这是一个属于建筑改造的项目案例，是把集装箱经过改造之后同原建筑结合在一起，总改造面积为42平米。

从建筑外部看到的效果是原建筑旁边放着一个充满涂鸦的集装箱，二者看似是没有联系的，但爱建筑内部则完全不同，建筑师把两者的设计紧密的结合在一起，犹如一个整体。

工作室的墙壁也摆放了一幅色彩斑斓的抽象图画，这与集装箱的外墙涂鸦相互呼应。

天花板上的吊灯颇具创意，整个工作室的设计如家居设计一样叔叔，少了传统办公室的严肃，多了一分悠闲。

无论是从室内设计还是室内的摆设都别具匠心，简洁优雅的办公环境能让工作者轻松地投入到工作之中，真是工作与享受两不误。
